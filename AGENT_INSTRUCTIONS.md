# Инструкции для AI-агента: Перевод NextCRM на русский

## Задача

Перевести CRM-приложение NextCRM (Next.js приложение) на русский язык.

## Основные правила поведения (Andrej Karpathy)

### 1. Думай перед кодом
- Не предполагай — спрашивай если непонятно
- Если есть несколько вари interpretation — представь их, не выбирай молча
- Если что-то непонятно — остановись и спроси

### 2. Простота
- Минимальный код который решает задачу
- Никаких абстракций для одноразового кода
- Если пишешь 200 строк а можно 50 — перепиши

### 3. Хирургические изменения
- Только то что нужно для задачи
- Не улучшай соседний код
- Не рефакторь working вещи

### 4. Целеориентированность
- Определяй критерии успеха
- Проверяй результат перед завершением

## Структура перевода

### Файлы перевода
1. **locales/ru.json** — основной файл переводов (уже существует, нужно дополнить)
2. Компоненты с хардкодом — искать английские строки в компонентах

### Что переводить
1. Все ключи в `locales/en.json` → `locales/ru.json`
2. Хардкод в React-компонентах:
   - Меню (CRM, Reports, Campaigns, Settings)
   - Кнопки и labels
   - Сообщения об ошибках
   - Формы (Forms)
3. Даты и числа на русский формат

### Как искать неперведенные строки
```bash
# Искать английские строки в компонентах
grep -r "\"[A-Z]" --include="*.tsx" app/
```

### Проверка перевода
1. Запустить приложение: `npm run dev`
2. Открыть http://localhost:3000/ru
3. Проверить все страницы

## Критерии успеха

- [ ] Вся CRM работает на русском
- [ ] Меню переведено
- [ ] Формы переведены
- [ ] Сообщения переведены
- [ ] Даты в российском формате (DD.MM.YYYY)
- [ ] Нет английских слов в интерфейсе
